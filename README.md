# AudioPlay
an Audio Player that runs in the Terminal uses the miniaudio library.
#  USAGE
### Start :   
```sh
  ./AudioPlay  [file].mp3
```  
  or   
  ```sh
  ./AudioPlay
```
  and it will ask you to enter an audio file path
    
### nav:
  - 'S' or 's' to stop
  - 'C' or 'c' to continue
  - 'a' and 'd' for seeking backward and forward in the timeline
  - 'E' for exiting
